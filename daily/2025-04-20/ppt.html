microsoft/BitNet

1 比特 LLM 官方推理框架

14.6k 1k 145 49 16 C++ 4天 4次 627
编程语言占比: C++ 52.7%, Python 46.5%, Other 0.8%
项目速读

BitNet 是微软为 1-bit 大语言模型 设计的高效推理框架，专攻 CPU 运行。通过优化内核，在 ARM/x86 CPU 上实现显著加速 (高达 5.07x) 和能耗降低 (高达 70%)。核心优势是高性能低功耗，使单 CPU 运行百亿参数模型成为可能。基于 llama.cpp，易集成，是探索超低比特 LLM 潜力的理想工具。
