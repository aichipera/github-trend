🌟 今日明星项目：onlook-dev/onlook

onlook-dev/onlook

专为设计师打造的光标 • 开源的可视化 Vibecoding 编辑器 • 利用 AI 可视化构建、设计样式并编辑 React 应用

16.3k ⭐ TypeScript 连续在榜: 6次 今日新增: 1477 ⭐

Onlook 弥合设计与代码之间的鸿沟，允许设计师和开发者在浏览器中通过直观的可视化界面构建、样式化和编辑 React 应用，所有操作实时同步到底层代码。其核心在于将强大的可视化编辑能力与真实的、可控的代码相结合，并利用 AI 提供辅助生成和编辑功能。

Aha Moment: 所见即所得的设计与可控代码的完美融合，加上AI辅助，极大提升Web开发效率！

Anduin2017/HowToCook

程序员在家做饭方法指南

87.8k ⭐ Dockerfile (内容组织) 总上榜: 8次 今日新增: 989 ⭐

🤖 深入AI Agent：frdel/agent-zero

frdel/agent-zero

Agent Zero AI 框架

8.6k ⭐ Python 连续在榜: 6次 今日新增: 321 ⭐

Agent Zero 是个人化 AI 代理框架，将操作系统作为工具，赋能代理理解和执行复杂任务 (软件开发, 系统管理, 内容创作)。其独特之处在于代理能自行编写和执行代码及终端命令，提升透明度、灵活性和可定制性。支持多智能体协作，提供实时交互界面。

nautechsystems/nautilus_trader

高性能算法交易平台以及事件驱动回测器

7.8k ⭐ Python / Rust 连续在榜: 5次 今日新增: 302 ⭐

NautilusTrader 是面向专业量化交易员的高性能、生产级开源平台。它解决了回测到实盘代码不一致的鸿沟，允许用同一套代码进行高精度回测并无缝部署生产。核心引擎Rust构建，确保极低交易延迟和强大稳定性，同时提供Python接口，简化开发。
