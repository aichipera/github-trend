frdel/agent-zero

Agent Zero AI框架

Python 9.2k 2239

项目速读

Agent Zero 是一个个人用的、有机生长的智能代理框架，致力于成为能与用户一同学习和进化的通用型个人助手。核心理念是将计算机本身视为完成任务的工具，通过自主执行代码和命令，并支持多层级代理协作来解决复杂问题。关键价值在于具备持久记忆、高度可定制性、透明交互以及利用操作系统能力的动态适应性。

nautechsystems/nautilus_trader

高性能算法交易平台

Python & Rust 8.3k 2193

项目速读

NautilusTrader 是一个为专业人士打造的高性能量化交易平台和事件驱动回测系统。它核心解决了量化策略在研究回测与生产实盘环境之间的一致性问题，并提供了极致的性能。平台的核心引擎由 Rust 构建以确保速度与稳定性，同时通过高性能 Python 接口提供友好的开发体验。

anthropics/prompt-eng-interactive-tutorial

Anthropic 的交互式提示工程教程

Jupyter 11.2k 4546

项目速读

Anthropic 官方推出的交互式提示词工程教程，旨在系统地教授如何为 Claude 系列大模型编写和优化提示词。核心亮点在于其强调实践和高度交互性，提供丰富的案例和练习，并内置“Example Playground”区域，允许用户直接在文档中修改提示词并即时查看 Claude 的响应。

donnemartin/system-design-primer

大规模系统设计指南

Python 305k 3314

项目速读

一个关于大型系统设计的宝贵资源库，旨在帮助开发者系统地学习如何设计可扩展、高可用、高性能的复杂系统，并为严苛的系统设计面试做好充分准备。核心优势在于内容的全面性和实用性，收录了大量常见的系统设计面试问题，并附带详细的解题思路和图示。

Fosowl/agenticSeek

完全本地运行的 Manus AI

Python 17.4k 3349

项目速读

一个开源的、完全本地运行的 AI 智能代理项目，旨在为用户提供一个强大的、无需依赖昂贵云服务或外部 API 的自主助手。它解决了传统 AI 代理服务成本高昂和数据隐私泄露的问题。能够像真人一样思考、自主浏览互联网获取信息、理解并执行复杂指令，甚至能作为编程助手。
